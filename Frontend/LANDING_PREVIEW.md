## Color Palette

### Primary Colors
- **Primary Green**: `#25D366` - WhatsApp brand color
- **Secondary Teal**: `#128C7E` - Accent color
- **Neutral Gray**: `#F9FAFB` to `#111827` - Background/text

### Semantic Colors
- **Success**: `#10B981` - Positive actions
- **Warning**: `#F59E0B` - Alerts/highlights
- **Error**: `#EF4444` - Errors/destructive actions
- **Info**: `#3B82F6` - Information

### Gradients
- Hero background: `from-primary-50 via-white to-white`
- CTA button: `from-primary-600 to-secondary-600`
- Animated blobs: `primary-200/30` and `secondary-200/30`

## Typography

### Font Family
- **Inter** - Google Fonts (variable font)

### Hierarchy
- **H1**: 4xl/5xl/6xl - Main headline
- **H2**: 3xl/4xl/5xl - Section headers
- **H3**: xl/2xl - Card titles
- **Body**: base/lg - Descriptions
- **Small**: sm/xs - Labels, captions

### Font Weights
- **Bold** (700): Headlines, CTA buttons
- **Semibold** (600): Subheadings, labels
- **Medium** (500): Body text emphasis
- **Regular** (400): Body text

## Spacing System

### Section Padding
- Mobile: `py-20` (5rem)
- Desktop: `py-32` (8rem)

### Grid Gaps
- Mobile: `gap-4` to `gap-8`
- Desktop: `gap-8` to `gap-10`

### Container Max Width
- `max-w-7xl` (1280px) for all sections
- Centered with horizontal padding

## Animation Timings

### Transitions
- **Fast**: 0.2s - Hover effects, color changes
- **Normal**: 0.3-0.5s - Slide/fade animations
- **Slow**: 0.8-1.5s - Background effects, connecting lines

### Delays
- Staggered: 0.1s increments for list items
- Sequential: 0.2s increments for sections

### Easing
- **easeInOut**: Default smooth transitions
- **easeOut**: Entry animations

## Responsive Behavior

### Mobile (< 768px)
- Single column layouts
- Stacked cards
- Full-width buttons
- Reduced text sizes
